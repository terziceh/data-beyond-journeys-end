## What This Episode Is

This is a **quick filler episode** where I experimented with building a simple AI agent using ChatGPT’s custom GPT builder.

The goal was to prototype a system that could generate **personalized weekly meal plans** based on:

- fitness goals
- body composition targets
- macro needs
- food preferences

This is not the full version of the project yet, but it demonstrates the **initial concept.**

---

## What the Agent Does

The GPT can generate structured meal plans by considering:

- calorie goals
- protein intake
- macro balance
- food preferences
- dietary restrictions

Example output includes:

- breakfast
- lunch
- dinner
- snacks
- estimated macros

---

## Why I Built This

One theme of this repository is exploring **AI agents for real-world decision making**.

Nutrition planning is a great example of a **multi-variable optimization problem**, involving:

- calories
- macros
- taste preferences
- health constraints
- convenience

This experiment was a quick prototype to see how well an AI agent could handle this type of planning task.

---

## Lessons Learned

A few interesting takeaways:

- AI works well as a planning assistant when given clear constraints
- System instructions are extremely important
- Even simple agents can produce surprisingly useful outputs
