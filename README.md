### Episode 5 — Movie Recommendation System (Prototype)

Built an end-to-end recommendation system using the MovieLens dataset.

This iteration moved beyond a mockup into a working pipeline:

* Engineered user-level and movie-level features
* Trained a regression model to predict user ratings
* Generated recommendations for both simulated and real users
* Evaluated outputs against real user behavior

Key takeaway:

> A working recommender system is not just about predicting ratings — it's about understanding how user behavior, content features, and system design interact.

This version serves as a **baseline system**, highlighting both the strengths of simple models and the limitations of popularity-driven recommendations.
